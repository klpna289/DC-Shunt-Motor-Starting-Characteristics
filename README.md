# DC-Shunt-Motor-Starting-Characteristics (MATLAB/Simulink)
Simulation of DC shunt motor starting characteristics using MATLAB/Simulink with step-wise starting resistance to limit inrush current. (MATLAB/Simulink)

## Objective
To study and simulate the starting characteristics of a DC shunt motor using MATLAB/Simulink.

## Tools Used
- MATLAB
- Simulink

## Description
This project models a DC shunt motor in Simulink to analyze its starting behavior.  
At starting, back EMF is zero, causing high armature current.  
To limit this, step-wise starting resistances are used and gradually removed.

## Components Used
- DC Voltage Source
- DC Shunt Motor Model
- Starting Resistances (R1–R5)
- Circuit Breakers / Switches
- Measurement Blocks
- Scope (for output graphs)
